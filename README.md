# Aurora Task Tracker

Aurora Task Tracker is a lightweight, Markdown‑first productivity app designed for makers, developers, and writers who live in their notes. It lets you manage tasks, projects, and daily logs using plain text files that sync across all your devices.

## Features

- **Markdown‑native:** Write tasks, notes, and project docs in the same place using standard Markdown.
- Smart task parsing: Turn `- [ ]` list items into trackable tasks with due dates and tags.
- Daily notes: Automatic creation of dated journals (e.g. `2026-04-02.md`) for quick capture.
- Cross‑project search: Instantly find tasks, decisions, and notes across your entire workspace.
- Local‑first storage: Your data lives in your filesystem; Aurora just adds structure on top.
